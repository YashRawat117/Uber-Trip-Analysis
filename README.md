# Uber-Trip-Analysis-Dashboard
Built an Uber Trip Data Analysis dashboard in Power BI to analyze trip volume, revenue, peak hours, trip distance, and location trends. Cleaned and transformed data, created interactive visuals, KPIs, and filters to identify demand patterns, customer behavior, and operational insights for better decision-making. The project focuses on transforming raw trip data into actionable business insights through visual analytics.

## Objectives
- Analyze trip performance and booking trends  
- Monitor revenue and trip activity  
- Identify peak demand hours  
- Understand trip distance and location patterns  
- Support data-driven decision making  

## Tools & Technologies
- Power BI  
- Power Query  
- DAX  
- Data Modeling  
- Data Cleaning  
- Interactive Dashboards  

## Key Metrics
- Total Trips  
- Total Revenue  
- Average Trip Distance  
- Peak Booking Hours  
- Pickup / Drop-off Trends  
- Trip Status Analysis  

## Dashboard Features
- Interactive filters and slicers  
- KPI cards for quick insights  
- Time-based trend analysis  
- Trip distance distribution visuals  
- Location-wise trip demand analysis  
- Clean and user-friendly dashboard layout  

## Key Insights
- Identified busiest booking hours  
- Tracked revenue trends over time  
- Analyzed popular pickup and drop locations  
- Measured average trip distance patterns  
- Highlighted demand fluctuations across periods  

## Challenges Addressed
- Cleaned inconsistent raw data  
- Built relationships between tables  
- Optimized DAX measures for KPIs  
- Improved dashboard performance and usability
